## [003] 关于 Domain Page 的标题部分优化

### 功能

所有 response_type: page 的页面标题（包含 Home 的页面）

### 状态

- 完成时间：

### 现状

- 部分 Page 没有顶部 banner

### 改进

- 为了美观，每个 Page 提供一个顶部Banner
- Banner内容包括：
  - 相关的背景图片，可选择的图片在 public/banner*.png 文件，不同 Domain 的page有多个图片，每次刷新随机选择
  - Home 的Banner图片使用 banner-lifeware.png
- Banner样式
  - 宽度自适应（图片内容等比例放大/缩小）
  - 高度固定
  - banner 高度固定为 80 px

- 每个Page Banner 下要显示标题
  - Home Page 显示“我的时间盒”
  - 其他的 Page 显示 action 的名称

# 主题：习惯管理优化迭代



### [004] AI助手界面优化

### 功能

AI助手

### 状态

- 完成时间：

### 现状

- 新对话界面不美观，与大多数用户习惯的 LLM 新对话界面不一致
- 点击“常用意图”，则直接执行，这没有任何意义
- 如果用户反复点击新对话而没有输入实质性内容，系统会不断增加对话记录，如果我在已有对话界面中点击“新对话”，主显示界面不会刷新到新的空白对话

### 修复意见

1. 新对话界面中
   - 常用提示选项放在对话框下面，现在是放在对话框上面，体验比较奇怪，同时常用提示选项可以加上快捷方式，
   - 界面内容的位置偏下，上面空白太多，可以上移一些
   - 附件一般是放在对话框内，现在对话框外

2. 常用意图修改
   - 常用意图显示，应显示“意图名称+快捷方式”，点击后，不是直接提交，而是将“/快捷方式” 直接显示到对话框中，并定位输入焦点，让用户继续输入其他信息后才提交。

3. 新对话的修改
   - 在已有对话中点击“新对话”，重新显示“新对话”界面
   - 新对话未进入实质对话时，再点击“新对话”不需要执行任何操作

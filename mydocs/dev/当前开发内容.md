### [014] 查询习惯统计

### 功能

“成长领域”--“习惯”--“查询习惯统计”

### 状态

- 完成时间：



### 现状

- 未开发



### 需求

- 是一个 view_route

- 分为日/周/月三个视图（与Timebox的视图类似）

- 日视图：

  - 显示如下视图

    - | 习惯       | 状态          | 近5天       |
      | ---------- | ------------- | ----------- |
      | 时间盒计划 | ✅ 连续6天     | ✅✅✅✅✅       |
      | 每日总结   | ✅ 连续6天     | ✅✅✅✅✅       |
      | 运动       | ❌ 中断1天     | ❌✅✅✅✅ (4/5) |
      | 读书笔记   | ❌ 连续中断2天 | ❌❌✅✅❌ (2/5) |
      | 产品修炼   | ❌ 连续中断2天 | ❌❌❌✅❌ (1/5) |

  - 可点击习惯详情显示，包括历史打卡详情

  - 右边栏是本月日历（与Timebox类似）

- 周视图

  - 显示周一至周日
  - 每周下面显示打卡信息

- 月视图

  - 显示该日打卡的习惯
